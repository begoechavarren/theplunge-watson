# Cambio Bego

This Node.js app demonstrates the Watson Assistant service in a simple chat interface simulating a cognitive car dashboard.

## Adapted from the Plunge
Cloned from https://github.com/watson-developer-cloud/assistant-simple and modified to run chatbot for The Plunge.

Command line: $npm start
localhost:3000
